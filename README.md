# アップルワールド羽村店 LP

西多摩エリアの中古車買取「アップルワールド羽村店」の問い合わせ獲得型ランディングページです。
HTML / CSS / 素のJavaScript のみ。ビルド不要で `index.html` を開けば動きます。

## ファイル構成

```
index.html              … 本体（全セクション）
assets/css/style.css     … スタイル（モバイルファースト / レスポンシブ）
assets/js/main.js        … FAQアコーディオン・スクロール追従CTA
```

## 公開前に差し替える項目（重要）

本番公開の前に、以下の「仮の値」を実際の情報へ置き換えてください。

| 箇所 | 仮の値 | 差し替え先 |
|------|--------|-----------|
| 電話番号 | `0120-000-000` / `tel:0120000000` | 実際のフリーダイヤル・電話番号 |
| LINE | `https://line.me/` | 公式LINEの友だち追加URL |
| 古物商許可番号 | `第000000000000号` | 実際の許可番号 |
| 住所・アクセス・営業時間 | 仮表記 | 実際の店舗情報（フッター内） |
| canonical / OGP URL | `https://www.example.com/` | 公開ドメイン |
| 問い合わせフォーム | `action="#"` | フォーム送信先（メール送信スクリプト or フォームサービス） |

電話番号は `index.html` 内に複数箇所あります。エディタの一括置換で
`0120-000-000`（表示用）と `tel:0120000000`（リンク用）をまとめて置換すると確実です。

## 画像の差し替え

写真はすべて「プレースホルダー（色付きの枠）」で実装しています。実画像に差し替える箇所：

- **ファーストビューの店長写真** … `.hero__photo-img`（推奨 縦長 900×1100px 程度）
- **店長プロフィール写真** … `.manager__photo-img`（推奨 縦長 800×1000px 程度）
- **対象車種カード / 買取実績カード** … `.target__img` / `.results__img`

実画像にする場合は、該当の `<div ... role="img">` を `<img src="..." alt="...">` に置き換えるか、
CSS の `background` に写真を指定してください。`alt` には地域名＋車種など
（例「羽村市 ハイエース 買取実績」）を入れるとSEOに有効です。

## 対応エリアマップ

`.area__map` は擬似的なMAP表現です。公開時はコメント箇所を
Googleマップのiframe（埋め込み）に置き換えると実用的です。

## 設計メモ

- スマホ最優先・レスポンシブ（768px / 900px ブレイクポイント）
- CTA（電話・LINE・フォーム）を全画面で常時アクセス可能
- スクロール追従CTA：ファーストビューを過ぎると下部に出現（スマホ）/ 右下フロート（PC）
- 外部依存は Google Fonts のみ。表示速度を重視し、画像は軽量なCSS表現
- `prefers-reduced-motion` 対応
